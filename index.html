Redesign the File Sharing Experience for SAP Digital UX Team

The work documents of SAP Digital UX Team need to migrate from an OneDrive shared folder to a new SharePoint site. I utilized UX design process to planned the file migration process. I conducted interviews with the stakeholders to define needs and goals, planned and prototyped the site's structure, finally designed and implemented the pages on SharePoint.
